Introducción

ATDF es un protocolo abierto para la descripción funcional de herramientas, que permite a los agentes de IA seleccionarlas y utilizarlas basándose en su propósito, contexto y operación, sin depender de detalles técnicos específicos.

ATDF estandariza las descripciones de herramientas en torno a tres preguntas fundamentales:

  • ¿Qué hace la herramienta?
  • ¿Cuándo debe utilizarse?
  • ¿Cómo se utiliza?

Este enfoque funcional permite a los agentes de IA seleccionar herramientas basadas en la tarea en cuestión, en lugar de requerir conocimientos específicos sobre nombres o APIs de herramientas.

Versiones

ATDF está disponible en dos versiones compatibles:

0.1.0 Básica

  • Funcionalidad principal con campos esenciales
  • Estructura simple para descripciones básicas
  • Perfecta para empezar o casos de uso simples

0.2.0 Mejorada

  • Toda la funcionalidad básica más características avanzadas opcionales
  • Soporte multilingüe integrado
  • Prerrequisitos y dependencias
  • Ejemplos de uso con salidas esperadas
  • Indicadores de progreso y finalización
  • Completamente compatible con v0.1.0

Características

Formato simple y legible

JSON/YAML estructurado que es fácil de entender tanto para humanos como para agentes de IA.

Agnóstico de modelo

Funciona con cualquier modelo de agente de IA, independientemente de su arquitectura.

Agnóstico de herramientas

Describe tanto herramientas físicas (p.ej., taladros) como digitales (p.ej., APIs).

Soporte multilingüe

Soporte para descripciones de herramientas en múltiples idiomas (español, inglés, portugués y más).

Metadatos

Organiza herramientas con información de versión, autor, etiquetas y categoría.

Ejemplos de uso

Incluye ejemplos reales que ayudan a los agentes a entender cómo usar las herramientas.

Ejemplo de una herramienta básica que crea agujeros:

{
  "tool_id": "hole_maker_v1",
  "description": "Permite crear agujeros en superficies sólidas",
  "when_to_use": "Usar cuando necesites generar un agujero en una pared",
  "how_to_use": {
    "inputs": [
      { "name": "location", "type": "string", "description": "Ubicación del agujero" },
      { "name": "bit_id", "type": "string", "description": "ID de la broca" }
    ],
    "outputs": {
      "success": "Agujero creado con éxito",
      "failure": [
        { "code": "invalid_bit", "description": "Broca no compatible" }
      ]
    }
  }
}

Ejemplo de una herramienta mejorada con características adicionales:

{
  "tool_id": "enhanced_hole_maker_v1",
  "metadata": {
    "version": "1.0.0",
    "author": "ATDF Development Team",
    "tags": ["física", "perforación", "construcción"],
    "category": "herramientas_físicas",
    "created_at": "2025-06-15",
    "updated_at": "2025-06-15"
  },
  "localization": {
    "es": {
      "description": "Permite crear agujeros en diversos tipos de superficies",
      "when_to_use": "Usar cuando necesites realizar perforaciones precisas"
    },
    "en": {
      "description": "Creates holes in various types of surfaces",
      "when_to_use": "Use when you need to make precise holes"
    }
  },
  "description": "Permite crear agujeros en diversos tipos de superficies",
  "when_to_use": "Usar cuando necesites realizar perforaciones precisas",
  "prerequisites": {
    "tools": ["superficie_marcada_v1", "nivel_burbuja_v2"],
    "conditions": ["superficie_seca", "no_circulacion_electrica"],
    "permissions": ["manipulacion_estructural"]
  },
  "how_to_use": {
    "inputs": [
      { "name": "location", "type": "string", "description": "Ubicación del agujero" },
      {
        "name": "configuration",
        "type": "object",
        "schema": {
          "properties": {
            "depth": {"type": "number", "description": "Profundidad en mm"},
            "diameter": {"type": "number", "description": "Diámetro en mm"}
          },
          "required": ["depth", "diameter"]
        },
        "description": "Configuración del agujero"
      }
    ],
    "outputs": {
      "success": "Agujero creado con éxito",
      "failure": [
        { "code": "invalid_configuration", "description": "Configuración no compatible" }
      ]
    }
  },
  "feedback": {
    "progress_indicators": ["vibracion", "temperatura"],
    "completion_signals": ["profundidad_alcanzada"]
  },
  "examples": [
    {
      "title": "Perforar pared de yeso",
      "description": "Ejemplo básico de perforación",
      "inputs": {
        "location": "x:120,y:85",
        "configuration": {
          "depth": 35,
          "diameter": 8
        }
      },
      "expected_output": "Agujero creado con éxito"
    }
  ]
}

Para comenzar a usar ATDF, puedes:

  1. Explorar los ejemplos para entender el formato.
  2. Crear tus propias descripciones de herramientas siguiendo la especificación.
  3. Validar tus herramientas con el validador:
python tools/validator.py path/to/your/tool.json

Prueba la demostración para ver ATDF en acción:

python tools/demo/atdf_showcase.py
